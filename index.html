Web Scrapping project


This Django project, developed by Parham Moeini, is an apartment search portal that allows users to search for rental listings in various cities. The project leverages web scraping techniques to gather real estate data from platforms like Zumper and Apartments.com.
The extracted information, such as pricing, address, and contact details, is presented to users in an organized and user-friendly format on the website. After fetching all the data, the program also produces a Excel file for each sourse and let user download the file.
The project includes features like, login/sign up system, user authentication, seamless search functionality, and dynamic data display, making it easy for users to find available apartments based on city input.
